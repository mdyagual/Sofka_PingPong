PONG (1972)

Para iniciar/pausar el juego, se debe presionar la barra espaciadora. Si desea reiniciar el juego, presione F5

Teclas:

Barra izquierda: Arriba ( W ) | Abajo ( S ) ---- Barra derecha: Arriba ( ↑ ) | Abajo ( ↓ )

Limitaciones:
-La pelota solo rebota contra las barras. A futuro se puede mejorar para que rebote en el borde superior e inferior.
-Se pueden hacer mejoras estéticas para recrear el estilo del juego original.
